This option is used to specify the type of interlacing scheme for raw image formats such as RGB or GRAY. Choose from these interlace types:

None
do not interlace (RGBRGBRGBRGBRGBRGB...).

Line
use scanline interlacing (RRR...GGG...BBB...RRR...GGG...BBB...).

Plane
use plane interlacing (RRRRRR...GGGGGG...BBBBBB...).

Partition
like plane interlacing except the different planes are saved to individual files (e.g. image.R, image.G, and image.B).

The default scheme is None.

Use Line, or Plane to create an interlaced GIF or progressive JPEG image.
